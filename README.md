# 도매시장 일일 분석 (Daily Wholesale Market Analysis)

전국 32개 공영도매시장, 83개 법인/공판장의 정산 데이터를 매일 자동 수집하고, AI 분석 리포트를 생성합니다.

## 데이터 소스

### 정산정보 (katSale) — 핵심 수집 파이프라인

- **API**: [data.go.kr](https://www.data.go.kr/) — 한국농수산식품유통공사 정산정보
- **엔드포인트**: `https://apis.data.go.kr/B552845/katSale/trades`
- **기준**: 정산일(`trd_clcln_ymd`) = 경매 후 취소 건 제외, 확정된 거래만 집계
- **규모**: 하루 약 1만 집계행 (32개 시장 합산)
- **범위**: 2018-01-03 ~ 현재

### 전자송품장 출하예약 (katElectronicInvoice2) — 수급 예측용

- **엔드포인트**: `https://apis.data.go.kr/B552845/katElectronicInvoice2/shipmentReservations2`
- **기준**: 출하예정일(`spmt_dt`) = 경매 전 사전 등록된 출하예약
- **용도**: 내일(+1일) 출하예정 물량으로 수급 예측
- **한계**: 전자송품장은 전체 출하량의 일부만 포함. 산지에서 전자송품장을 이용하지 않는 물량이 상당수 존재하므로, **시장 전체 입하량 대표성에 한계**가 있음

### 1순위 신규 API (개발 예정)

| API | 엔드포인트 | 데이터 출처 | 일일 건수 |
|-----|----------|-----------|----------|
| 출하량 추이 | `B552845/shipmentSequel/info` | 도매시장 실데이터 (법인별) | ~13,134 |
| 가격 추이 | `B552845/priceSequel/info` | KAMIS 중도매인 조사 | ~347 |
| 가격 등락 | `B552845/risesAndFalls/info` | KAMIS 중도매인 조사 | ~243 |
| 일별 도소매 | `B552845/perDay/price` | KAMIS | 미확인 |

> 출하량 추이는 전자송품장이 아닌 도매시장 법인별 실제 반입 데이터입니다.

## 구조

```
collect.py           — katSale 정산 API → 전국 32개 시장, 83개 법인/공판장 수집
collect_shipment.py  — 전자송품장 출하예약 수집 (내일 출하예정 물량)
analyze.py           — Gemini AI → 일일 분석 리포트 (마크다운)
compare.py           — 전일 대비 품목별 가격 변동 분석
djc_report.py        — 대전중앙청과 경영 분석 리포트 (HTML 이메일)
run_daily.py         — 통합 파이프라인 (보정 → 수집 → 출하예약 → 분석 → 비교)
backfill.py          — 과거 데이터 일괄 수집 (최근→과거, 스킵 기능)
backfill_new_markets.py — 기존 파일에 신규 시장 데이터 보충
send_email.py        — Gmail 이메일 자동 발송
send_telegram.py     — 텔레그램 자동 발송
```

## 일일 파이프라인 (run_daily.py)

1. **공판장 보정** — D-1, D-2 정산 데이터 재수집 (공판장 지연 반영)
2. **정산 수집** — 전일 정산 확정 데이터 (32시장 × 83법인)
3. **출하예약 수집** — 내일(+1일) 출하예정 물량
4. **AI 분석** — Gemini가 전체 시장 동향 분석
5. **전일 비교** — 품목별 가격 변동 자동 비교

## 사용법

```bash
# 환경변수 설정
export DATA_GO_KR_API_KEY=your_key
export GEMINI_API_KEY=your_key

# 의존성 설치
pip install -r requirements.txt

# 오늘 데이터 수집 + 분석 (전체 파이프라인)
python run_daily.py

# 특정 날짜
python run_daily.py --date 2026-03-09

# 개별 실행
python collect.py --date 2026-03-09
python collect_shipment.py --date 2026-03-20
python analyze.py --date 2026-03-09
python compare.py --today 2026-03-09 --prev 2026-03-06

# 과거 데이터 일괄 수집
python backfill.py --start 2020-01-10 --end 2018-01-03 --max 312
```

## GitHub Actions

매일 오전 8시(KST), 월~토 자동 실행.

- cron: `0 23 * * 0-5` (UTC) = KST 오전 8시, 월~토
- 전일(yesterday) 정산 데이터 수집 + 공판장 D-1/D-2 보정
- 출하예약 수집 + AI 분석 + 전일 비교
- DJC 경영 리포트 생성 + 이메일/텔레그램 발송
- 대시보드 데이터 자동 업데이트 (wholesale-dashboard)

Secrets:

| Secret | 용도 |
|--------|------|
| `DATA_GO_KR_API_KEY` | data.go.kr API 인증 |
| `GEMINI_API_KEY` | Google Gemini AI 분석 |
| `GMAIL_ADDRESS` | 이메일 발송자 주소 |
| `GMAIL_APP_PASSWORD` | Gmail 앱 비밀번호 |
| `TELEGRAM_BOT_TOKEN` | 텔레그램 봇 토큰 |
| `TELEGRAM_CHAT_ID` | 텔레그램 채팅방 ID |
